Let's assume Xe,Ye,Ze are the player eye coordinates and Xt,Yt,Zt are the texture point coordinates on some block nearby.
For each coordinate axis x,y and z we need to find which block planes the line e-t intersects and at which positions.

// all coordinates are float

For X:
float dX = Xt-Xe; // difference in coordinate X
float dY = Yt-Ye; // difference in coordinate Y
float dZ = Zt-Ze; // difference in coordinate Z

CheckX:

if (dX == 0) goto CheckY;

int xmin=0,xmax=-1;
if (dX > 0) {
  xmin = round(ceil(Xe));
  xmax = round(floor(Xt));
}
else if (dX < 0) {
  xmin = round(ceil(Xt));
  xmax = round(floor(Xe));
}

Then, if we loop the interger x from xmin to xmax (inclusive), we weill get all block plane x coords between the eye and the tex point.
Now we determine the y,z intesection points of the e-t line with the block planes:

float iY = dY/dX;
float iZ = dZ/dX;

for (int x = xmin; x<max; x++) {
  y = iY * (x-Xe) + Ye;
  z = iZ * (x-Xe) + Ze;

  //Now we need to be careful to calculate correct block coordinates
  //always round toward -Inf
  Y = (int)floor(y);
  Z = (int)floor(z);
}
